## 🎨 Design System

### Typography
- **Header**: Segoe UI, Bold, 26px
- **Subheader**: Segoe UI, Bold, 18px
- **Button Text**: Segoe UI, Bold, 14px
- **Body Text**: Segoe UI, Regular, 14px
- **Labels**: Segoe UI, Regular, 13px

### Color Palette
```
Primary (Maroon):      #6C1324 - Main actions & borders
Accent (Gold):         #F3C623 - Highlights & text on primary
Background (Off-White): #FAF5E6 - Main background
Text (Dark):           #1A1A1A - Primary text
Light BG:              #F9F7F0 - Secondary backgrounds
Border (Gold):         #D4AF37 - Accent borders
Success:               #27AE60 - Positive feedback
Error:                 #E74C3C - Error messages
```

### Spacing & Layout
- Panel borders: 15-20px
- Component spacing: 10-15px
- Table row height: 30px
- Button padding: 10-20px (height) × 20px (width)
